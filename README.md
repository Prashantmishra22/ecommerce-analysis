# 📊 E-Commerce User Behavior Analysis

## 🔹 Introduction

With the rapid growth of e-commerce platforms, understanding user behavior has become essential for improving customer experience and increasing sales. Every user interaction on a website—such as browsing, clicking, and purchasing—generates valuable data that can be analyzed to gain meaningful insights.

This project focuses on analyzing user behavior using data science techniques. By exploring patterns in user activity, the project aims to understand the factors that influence purchase decisions and user engagement.


## 🔹 Project Description

This project, titled **“Comprehensive E-Commerce User Behavior Analysis: Trends, Patterns, Engagement and Predictive Modeling Study”**, involves analyzing an e-commerce dataset using Python.

The dataset contains information about users such as age, gender, session time, pages viewed, engagement metrics, and purchase behavior.

The project includes:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization using graphs
* Hypothesis testing (t-test)
* Linear regression analysis

The main objective of this project is to identify patterns in user behavior and understand how different factors affect purchase decisions.

## 🔹 Key Insights

* Returning users are more likely to purchase
* Gender does not significantly affect purchase decisions
* Purchase behavior depends on multiple factors
